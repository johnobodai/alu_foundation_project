HTML Course - Lesson 2: HTML Basics

HTML Basics Examples

In this chapter we will show some basic HTML examples.

Don't worry if we use tags you have not learned about yet as you will get to learn them in subsequent lessons.

HTML Documents

An HTML document consists of elements that define the structure and content of a web page. Elements are enclosed in tags, and they provide meaning to the content they enclose.

All HTML documents must start with a document type declaration: <!DOCTYPE html>.

The basic structure of an HTML document includes the opening <html> tag and the closing </html> tag that contains the entire content of the page.

The visible part of the HTML document is between <body> and </body>

The <!DOCTYPE> Declaration

The <!DOCTYPE> declaration specifies the HTML version being used. It helps browsers understand the document's structure and rendering. For HTML5, the <!DOCTYPE> declaration is simplified to <!DOCTYPE html>.

It must only appear once, at the top of the page (before any HTML tags).

The <!DOCTYPE> declaration is not case sensitive.

HTML Headings

HTML provides six levels of headings, from <h1> to <h6>, which represent varying levels of importance. Headings help structure the content and provide visual hierarchy on the page.

<h1> defines the most important heading. <h6> defines the least important heading.

Paragraphs

The <p> tag is used to define paragraphs in HTML. Paragraphs group related text together and provide better readability.

Links

Links, also known as hyperlinks, are used to navigate between different web pages. The <a> tag is used to create links, and the href attribute specifies the URL of the target page.

Images

The <img> tag is used to display images on a web page. The src attribute specifies the image source, and the alt attribute provides alternative text for accessibility.
